bilibili-hyperlinks
 
b站内，发任意网站链接，的简易方法 2026年4月9日可用
 
B 站 game_jump 跳转链接生成模板
 
通用模板
 
http://d.bilibili.com/?preUrl=bilibili://game_center/game_jump?url=[链接]&schema=bilibili://game_center/game_jump?url=[链接]&share_from=h5
 
备注
 
-  [链接]  直接填写原始网址即可，无需 URL 编码
- 两处  [链接]  必须填写相同的目标地址
 
示例（百度）
 
原链接： https://www.baidu.com 
 
生成后完整链接
 
http://d.bilibili.com/?preUrl=bilibili://game_center/game_jump?url=https://www.baidu.com&schema=bilibili://game_center/game_jump?url=https://www.baidu.com&share_from=h5
