# bilibili-hyperlinks
b站内，发任意网站链接，的简易方法 2026年4月9日可用

B 站 game_jump 跳转链接生成模板
 
通用模板
 
 http://d.bilibili.com/?preUrl=bilibili%3A%2F%2Fgame_center%2Fgame_jump%3Furl%3D[URL编码后链接]&schema=bilibili%3A%2F%2Fgame_center%2Fgame_jump%3Furl%3D[URL编码后链接]&share_from=h5
 
备注
 
-  `[URL编码后链接]`  需替换为经过 URL 编码的目标地址
 
示例（百度）
 
原链接： `https://www.baidu.com` 
编码后： `https%3A%2F%2Fwww.baidu.com` 
 
生成后完整链接
 
 http://d.bilibili.com/?preUrl=bilibili%3A%2F%2Fgame_center%2Fgame_jump%3Furl%3Dhttps%3A%2F%2Fwww.baidu.com&schema=bilibili%3A%2F%2Fgame_center%2Fgame_jump%3Furl%3Dhttps%3A%2F%2Fwww.baidu.com&share_from=h5
